Web Application for Mapping & Depth Classification of Water Bodies

This project focuses on analyzing and interpreting water bodies using modern AI-driven computer vision and machine learning techniques. The system processes satellite imagery to identify water regions, segment their boundaries, and classify approximate depth levels based on RGB characteristics.

Built using a combination of YOLOv8 segmentation and Logistic Regression classification, the platform enables accurate detection and polygon-based segmentation of water bodies. Integrated Google Earth Engine support allows users to explore long-term historical changes, enabling temporal analysis and environmental monitoring.

Designed as an interactive web application using Streamlit and OpenCV, the solution provides researchers, students, and planners with real-time visual insights to support climate studies, water resource management strategies, and seasonal environmental assessments.

Project Information

  • Type Academic Final Year Project
  • Project Date Oct 2024 – Apr 2025
