Football Predictions Using Models

This project develops a machine-learning pipeline to predict football match outcomes from historical match data. The system focuses on extracting meaningful features from sporting datasets and leveraging a classic, explainable model to demonstrate predictive capability and interpretability.

Implemented in Python with Scikit-learn, the pipeline includes thorough preprocessing, feature selection, and KNN optimization. Emphasis was placed on cleaning the data, tuning hyperparameters, and selecting features that improve model generalization and reduce overfitting.

Completed as an academic project, the model achieved high reliability with a reported accuracy of 96%. The work highlights how data-driven approaches can support sports analytics, strategy, and forecasting with straightforward, well-tuned machine learning techniques.
